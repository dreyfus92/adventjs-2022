# Advent JS 2022

This is a repository for [Advent of Code 2022](https://adventjs.dev/) solutions in JavaScript.

## How to install Deno?
```sh
curl -fsSL https://deno.land/x/install/install.sh | sh
```

## Checking Deno is installed
```sh
deno --version
```

## How to run
```ts
deno run --allow-read --watch ./[day]/a.ts
```
